# 📊 Task 1 — Data Immersion & Wrangling

## 🎯 Objective
To rapidly acquaint with the dataset and master the critical first step 
of any analysis: acquiring, cleaning, and preparing data for analysis.

## 📁 Files in this folder
| File | Description |
|------|-------------|
| `ApexPlanetTask_1.ipynb` | Main cleaning script — full Python code |
| `task1_cleaned_superstore.csv` | Final cleaned & analysis-ready dataset |

## 🔧 Steps Performed

### 1. Data Access & Familiarization
- Explored the Superstore dataset
- Created data dictionary documenting each variable's meaning and type

### 2. Data Quality Assessment
- Identified missing values, duplicates
- Detected inconsistent formatting and outliers

### 3. Data Cleaning & Transformation
- Handled missing values using Pandas
- Removed duplicate records
- Standardized date formats
- Feature engineering — created new derived columns
- Output final analysis-ready cleaned dataset

## 🛠️ Tools Used
- Python, Pandas, NumPy
- Google Colab / Jupyter Notebook

## 📌 Key Learnings
- Real-world data always has quality issues
- Data cleaning is 70% of any analysis project
- Proper documentation (data dictionary) is essential
