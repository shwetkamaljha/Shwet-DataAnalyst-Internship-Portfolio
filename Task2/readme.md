# 📈 Task 2 — Exploratory Data Analysis (EDA) & Business Intelligence

## 🎯 Objective
To uncover patterns, trends, and relationships within the data and 
develop proficiency in SQL for data extraction and basic dashboarding.

## 📁 Files in this folder
| File | Description |
|------|-------------|
| `ApexPlanetTask2.ipynb` | Complete EDA notebook with visualizations |

## 🔧 Steps Performed

### 1. Descriptive Statistics & Univariate Analysis
- Calculated key summary statistics (mean, median, std)
- Created histograms and bar charts to understand distributions

### 2. SQL for Business Questions
- Wrote SQL queries to answer 5-7 business questions
- Used filtering, aggregation, and multi-table joins

### 3. Multivariate Analysis & Correlation
- Created scatter plots, heatmaps, and pair plots
- Explored relationships between multiple variables

### 4. Static Dashboard Mock-up
- Built dashboard using Python (Matplotlib/Seaborn) + Power BI
- Proposed key KPIs to track for business decisions

## 🛠️ Tools Used
- Python, Pandas, Matplotlib, Seaborn
- Power BI
- SQL
- Google Colab / Jupyter Notebook

## 📌 Key Learnings
- EDA reveals hidden patterns before modeling
- SQL is powerful for quick business question answering
- Visualization makes insights accessible to non-technical stakeholders
